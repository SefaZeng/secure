## 研究现况
### 识别带噪音和干扰的验证码
去噪及抗干扰手段：图像二值化+开闭运算+腐蚀膨胀

处理字符粘结：骨架算法

验证码分割：连通域提取

识别：机器学习或深度学习都可以胜任
常见的用于图片识别领域的模型：SVM，CNN
